GameX Setup

Welcome to GameX, an interactive story adventure. This guide will help you get started with the project, whether you're playing or extending the game.

Quick Start

  1. Open the Game

    Simply click the "Play Now" button on the home page or navigate directly to game.html to start playing the game.

  2. Browser Requirements

    GameX works best with modern browsers like Chrome, Firefox, Edge, or Safari. JavaScript must be enabled.

  3. Game Controls

    Navigate through the story by clicking on the choice links at the bottom of the screen. Use the inventory button to access your items.

Project Resources

game.html - Main game file to launch the interactive story css/custom.css - Custom styling for the dual-pane layout js/dice-roller.js - Dice mechanics implementation js/inventory-system.js - Inventory management system js/combat-system.js - Turn-based combat mechanics js/sound-manager.js - Audio playback system js/gamex-utils.js - Utility functions js/gamex-core.js - Core game engine media/ - Directory containing game assets (images, audio)

Game Features

  • Character Creation - Create and customize your protagonist
  • Inventory System - Collect, use, and manage items throughout your adventure
  • Combat System - Engage in turn-based battles using dice mechanics
  • Branching Narratives - Your choices affect the story and outcome
  • Dice Roll Mechanics - Randomized elements for combat and skill checks
  • Sound Effects - Immersive audio for key moments
  • Dynamic Illustrations - Visual representation of scenes
  • Responsive Design - Play on desktop or mobile devices

For Developers

If you want to modify or extend the game:

  • Edit Story Content: Modify the passages in gamex-core.js to change or add to the story
  • Add Illustrations: Place new images in the media/images/ directory
  • Customize UI: Edit the CSS in css/custom.css to change the appearance
  • Add Game Mechanics: Extend the JavaScript modules in js/ directory
Important: The game saves progress to your browser's localStorage. Clearing your browser data will erase saved games.

Need Help?

If you encounter any issues or have questions:

  • Check the browser console (F12) for any error messages
  • Refer to the documentation in the docs/ directory
  • Examine the JavaScript files for implementation details
